# MySQL 变量定义与使用

## 1. 定义变量

### 用户变量 (会话级别)
```sql
SET @变量名 = 值;
-- 示例：
SET @user_count = 100;
SET @user_name = '张三';
```

### 局部变量 (存储过程/函数内)
```sql
DECLARE 变量名 数据类型 [DEFAULT 默认值];
-- 示例：
DECLARE total INT DEFAULT 0;
DECLARE user_email VARCHAR(100);
```

## 2. 使用变量

```sql
-- 查询变量值
SELECT @user_name;

-- 在SQL语句中使用
SELECT * FROM users WHERE id = @user_id;

-- 在存储过程中使用
SET total = total + 1;
```

## 3. 注意事项

1. 用户变量以`@`开头，在整个会话中有效
2. 局部变量只在定义的存储过程或函数内有效
3. 变量名区分大小写
4. 用户变量不需要预先声明
5. 局部变量必须指定数据类型
