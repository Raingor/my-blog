# 关系除法查找满足所有条件的记录

原sql:
```sql
SELECT * FROM products WHERE id IN ( SELECT product_id FROM products_relation WHERE cat_id IN (1, 2, 3) GROUP BY product_id HAVING COUNT(DISTINCT cat_id) = 3 );
```

对应的tp
```bash
 if (!empty($subCatIds)) {

            // 多对多关联查询：查找属于任意分类下的产品

            $relModel = new ProductsRelation();

            $catIds = explode(',', $subCatIds);

            $proIds = $relModel->field('product_id')

            ->group('product_id')

            ->having('COUNT(DISTINCT cat_id) = ' . count($catIds))

            ->having('SUM(cat_id IN (' . implode(',', $catIds) . ')) = ' . count($catIds))

            ->select()

            ->column('product_id');

            $where[] = ['id', 'in', $proIds];

        }
```

最优写法：
```sql
SELECT product_id FROM products_relation WHERE cat_id IN (1, 2, 3) GROUP BY product_id HAVING COUNT(DISTINCT cat_id) = 3;
```
