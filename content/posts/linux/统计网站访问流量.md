# 统计网站访问流量

### 使用可视化工具 `GoAccess`（强烈推荐）

安装 GoAccess：

```bash
# Ubuntu/Debian
sudo apt update && sudo apt install goaccess

# CentOS/RHEL
sudo yum install goaccess
```

生成 HTML 报告：

bash

深色版本

```nginx
goaccess /var/log/nginx/site1_access.log \
    --log-format=COMBINED \
    --date-format=%d/%b/%Y \
    --time-format=%H:%M:%S \
    -o /var/www/html/report_site1.html
```

然后浏览器访问：`http://your-server-ip/report_site1.html` 查看实时图表。
