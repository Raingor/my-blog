# 挂载云盘

- lsblk 
	- 查看云盘列表
- mount 
	- mkfs.ext4 /dev/vdb
	- 挂载云盘到 指定的地址
	- mount /dev/vdb /mnt


● lsblk ：查看云盘列表

● mount ：查看磁盘

○ mount /dev/vdb /mnt ： 手动挂载

cp /etc/fstab /etc/fstab.bak ： 备份 fstab

sudo sh -c "echo `sudo blkid /dev/vdb | awk '{print \$2}' | sed 's/\"//g'` /mnt ext4 defaults 0 0 >> /etc/fstab" ：写入 fstab（自动挂载）


mount -a 执行读取 `/etc/fstab` 找出里面所有标记为可挂载的条目 把当前还没挂载的全部挂载一遍
