# 定时器

### 编辑 root 用户的定时任务

bash

深色版本

```bash
sudo crontab -e
```

### 2. 添加以下行（每半小时执行一次）：

bash

深色版本

```bash
*/30 * * * * /usr/local/bin/monitor_sites.sh
```

> 这表示：每小时的第 0 分和第 30 分执行一次脚本。

### 3. 保存并退出：

- 按 `Esc`，输入 `:wq` 回车（vi/vim 编辑器）


## 验证定时任务是否生效

### 查看当前定时任务：

bash

深色版本

```bash
sudo crontab -l
```

你应该看到：

text

深色版本

```bash
*/30 * * * * /usr/local/bin/monitor_sites.sh
```

### 手动运行一次脚本测试：

bash

深色版本

```bash
sudo /usr/local/bin/monitor_sites.sh
```


通过指令 `which bash` 查看是否.sh里面第一行是放什么；
默认是 #!/bin/bash
通过指令查看后进行替换 `#!/usr/bin/bash`
