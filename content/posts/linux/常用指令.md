# 常用指令

- yes | cp -ru 【复制的文件的路径】 【复制到指定的路径】
	- yes 为 自动选yes 
	- cp -ru 迭代复制强制覆盖
- readlink -f backup_all_databases.sh  读取文件的全路径
- df -h 查看磁盘状态
- du -sh ./*   查看当前目录下的容量大小

## 压缩文件
- tar -czvf archive.tar.gz file1 file2 directory/      需要压缩的 文件 1，文件 2，目录一起压缩到 压缩文件
- tar -czvf foldername.tar.gz /path/to/folder/     压缩文件夹 到 指定压缩文件
	- `-c`：创建新归档
	- `-z`：使用 gzip 压缩
	- `-v`：显示过程（可选）
	- `-f`：指定文件名
- tar -xzvf archive.tar.gz -C /path/to/target/   解压到指定目录下

## 服务器之间传输文件
```bash
scp user1@server1:/path/to/source/file user2@server2:/path/to/destination/
```

### 统计目录下有多少个目录
```bash
ls -l | grep '^d' | wc -l
```

## DNS 查询
- `nslookup example.com`  查询域名的 DNS 解析结果（A 记录、CNAME 等）
- `nslookup -type=mx example.com`  查询域名的邮件交换记录
- `nslookup -type=ns example.com`  查询域名的 NS 记录
- `nslookup -type=txt example.com`  查询域名的 TXT 记录
- `nslookup 8.8.8.8`  反向解析 IP 对应的域名
