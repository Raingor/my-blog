### CSS

```css
.flex-layout {
    display: flex;
    gap: 20px;
}

.left-img img {
    display: none;
    width: 200px; /* Set appropriate dimensions */
    height: auto;
}

.left-img img.active {
    display: block;
}

.item-box {
    cursor: pointer;
    padding: 10px;
    border: 1px solid #ccc;
    margin-bottom: 10px;
    transition: background-color 0.3s;
}

.item-box:hover, .item-box.active {
    background-color: #f0f0f0;
}
```

### JavaScript

```javascript
$(document).ready(function () {
    // 监听右侧文字点击事件
    $(".item-box").on("click", function () {
        // 获取当前点击的索引
        const index = $(this).index();

        // 切换左侧图片
        $(".left-img img").removeClass("active").eq(index).addClass("active");

        // 切换右侧文字的激活状态
        $(".item-box").removeClass("active");
        $(this).addClass("active");
    });
});

```

### 功能说明

1. **图片切换**：
    
    - 所有图片初始状态为 `display: none`。
    - 点击右侧文字时，JavaScript 会将对应图片的 `class` 设置为 `active`，显示该图片并隐藏其他图片。
2. **文字激活状态**：
    
    - 使用 `.item-box.active` 设置点击后的样式反馈（如背景高亮）。
3. **动态交互**：
    
    - 根据右侧文字的 `data-index` 属性来对应左侧图片的索引。

### 效果预览

1. 页面加载后，默认显示第一张图片。
2. 点击右侧某个文字部分，左侧图片会相应切换，同时右侧被点击的文字块背景高亮。
3. 支持动态扩展图片和文字，只需保持 `data-index` 对应一致即可。
