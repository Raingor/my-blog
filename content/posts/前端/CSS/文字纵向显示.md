# 文字纵向显示

要实现文字**纵向显示**（即文字从上到下排列），可以使用 CSS 中的 `writing-mode` 属性。以下是常用的样式写法：

### 基本样式（竖排文字，从上到下，从右向左排列）：

```css
.vertical-text {
  writing-mode: vertical-rl; /* 竖排，从右向左 */
  text-orientation: mixed;   /* 控制字符方向，可选值：mixed, upright, sideways */
}
```

### 常见取值说明：

- **`writing-mode: vertical-rl;`**  
    文字垂直排列，列的方向是从右向左（Right to Left）。这是中文竖排的传统方式。
    
- **`writing-mode: vertical-lr;`**  
    文字垂直排列，列的方向是从左向右（Left to Right）。
    
- **`text-orientation`**：
    
    - `mixed`：保持字符原有方向（拉丁字母会侧躺）。
    - `upright`：所有字符正立显示（适合中英文混排）。
    - `sideways`：所有字符旋转90度（较少用）。

### 示例 HTML + CSS：

```html
<div class="vertical-text">纵向文字示例</div>
```

```css
.vertical-text {
  writing-mode: vertical-rl;
  text-orientation: upright;
  font-size: 24px;
  line-height: 1.5;
}
```

> 💡 提示：在中文排版中，通常使用 `writing-mode: vertical-rl` 配合 `text-orientation: upright`，以获得传统书籍那样的竖排效果。
