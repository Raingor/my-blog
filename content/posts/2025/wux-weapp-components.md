[wux-weapp](https://github.com/wux-weapp/wux-weapp) 是一款功能丰富的微信小程序组件库，提供了众多常用的 UI 组件。

## 特性

- **丰富组件**：按钮、表单、弹窗、列表等 20+ 组件
- **自定义样式**：支持主题定制
- **易用性**：简单的 API 设计
- **文档完善**：详细的使用文档

## 安装

```bash
npm install wux-weapp
```

## 常用组件

- `wux-button` 按钮
- `wux-dialog` 弹窗对话框
- `wux-toast` 消息提示
- `wux-loading` 加载动画
