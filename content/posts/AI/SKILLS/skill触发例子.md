# skill触发例子

好的，这个 `ro-cms-dev` Skill 有以下几种触发方式：

## 方式1：自然触发（最常用）

直接在对话中提到触发关键词，Skill 会自动激活：

### 触发关键词：
- **核心关键词**：`ro-cms`、`rocms`
- **功能关键词**：`cms开发`、`网站开发`、`模板迁移`、`后台开发`
- **业务关键词**：`企业官网`、`电商网站`、`犀利士`
- **功能模块关键词**：`三级联动`、`订单查询`、`在线留言`

### 触发示例：

#### 示例1：新项目开发
```bash
你: 我要开发一个礼来犀利士的网站，使用 ro-cms
AI: [自动加载 Skill] 好的，我将创建项目目录 L-礼来犀利士，并从 Git 克隆基板...
```

#### 示例2：特定功能开发
```bash
你: 我需要实现三级联动地址选择功能
AI: [自动加载 Skill] 好的，我将使用 RO-CMS 的三级联动模块...
```

#### 示例3：后台管理开发
```bash
你: 我要开发一个产品管理的后台功能
AI: [自动加载 Skill] 好的，我将按照 RO-CMS 后台管理模板创建...
```

## 方式2：直接指定 Skill

明确告诉我要使用哪个 Skill：

```bash
你: 使用 ro-cms-dev skill 开发一个产品展示网站
AI: [加载 Skill] 好的，我将按照 RO-CMS 开发流程进行...
```

## 方式3：引用项目类型

```bash
你: 我要做一个类似礼来犀利士的项目
AI: [自动识别并加载 Skill] 好的，我将参考礼来犀利士项目的开发流程...
```

## 方式4：提到项目结构

```bash
你: 我要从 Git 克隆 Ro-CMS 基板开始开发
AI: [自动加载 Skill] 好的，我将从 Git 仓库克隆基板...
```

## Skill 加载后的效果

一旦 Skill 被激活，我会：

1. **遵循标准流程**：
   - 创建项目目录（L-项目名）
   - 从 Git 克隆基板（website-拼音）
   - 配置数据库和字符集

2. **使用模板和规范**：
   - 使用 `templates/` 中的代码模板
   - 参考 `references/` 中的技术文档
   - 遵循数据库字符集规范（utf8mb4）
   - 应用移动端响应式适配

3. **自动同步**：
   - 新功能开发后自动同步到基板项目
   - 更新工作记忆库
